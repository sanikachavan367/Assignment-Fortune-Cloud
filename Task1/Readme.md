# Task 1 — Footprinting, Reconnaissance & Scanning
This assignment demonstrates the process of information gathering and reconnaissance in cybersecurity. The objective is to collect publicly available information about a target domain using various footprinting and scanning techniques. Tools such as WHOIS, nslookup, theHarvester, and recon-ng are used to identify domain details, DNS records, subdomains, email addresses, IP information, and other reconnaissance data.

# Objectives
1. Understand footprinting and reconnaissance concepts
2. Perform domain and DNS information gathering
3. Identify publicly available target information
4. Use cybersecurity reconnaissance tools effectively
5. Analyze collected information for security assessment

# Tools Used
1. WHOIS
2. nslookup
3. theHarvester
4. recon-ng
5. nmap

# Key Activities
1. Domain information lookup
2. DNS record analysis
3. Subdomain enumeration
4. Email harvesting
5. Open-source intelligence (OSINT) gathering

# Conclusion
This task provides practical experience in the initial phase of cybersecurity assessment and helps understand how attackers and security professionals gather target information before performing security testing.
